MagicBook
一款用于自动化创建书页绑定的插件
作者：御宅松鼠
邮箱：2694836731@qq.com

安装方法：
1、解压后，将MagicBook放在：文档/maya/scripts 中
2、将 MagicBook 文件夹放在：文档/maya/plug-ins 中
3、将DropToInstall.mll 拖到maya视口，或直接将代码复制到脚本编辑器中运行。

然后你可以右键新按钮，换上悠米的图标（图标在MagicBook文件夹里面）

installation method:
1. After decompression, put the MagicBook in: documents/maya/scripts
2. Put the MagicBook folder in: documents/maya/plug-ins
3. Drag DropToInstall.mll to the maya viewport, or directly copy the code to the script editor to run(mel).

Then you can right-click the new button and replace it with the Yuumi icon (the icon is in the MagicBook folder)
